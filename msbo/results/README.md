# Results

This directory stores output files generated by the optimization experiments.

## Directory structure

Results are organized as `{process}_{complexity}/{seed}/`, for example:

```
results/
└── standard_50_2/
    └── 0/
        ├── ground_truth.txt
        ├── msopt_EI_UCB_summary.json
        ├── msopt_EI_UCB_denoised.npz
        ├── msopt_EI_UCB_*.png
        ├── random_summary.json
        ├── random_denoised.npz
        ├── global_EI_summary.json
        ├── global_EI_denoised.npz
        └── ...
```

## File descriptions

| File | Description |
|------|-------------|
| `ground_truth.txt` | Optimal parameter values and objective value |
| `{optimizer}_summary.json` | Detailed optimization results |
| `{optimizer}_denoised.npz` | Denoised objective values |
| `{optimizer}_*.png` | Performance and regret plots |

## Note

This directory is populated by running `main.py`. See the top-level [README](../../README.md) for usage instructions.
